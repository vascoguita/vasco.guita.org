Owned automation, infrastructure, and observability programs spanning FPGA test labs for CERN's particle accelerators, large-scale storage services, and CERN's Open Hardware Repository.

### Highlights

- Built hardware-in-the-loop testing infrastructure for FPGA gateware, firmware, and drivers powering CERN's particle accelerators and colliders, and packaged Linux drivers, libraries, and tools for Debian and RPM.
- Developed large-scale storage services used by **30,000+ users**, provided L3 support, and implemented distributed tracing with OpenTelemetry, Jaeger, Elasticsearch, and Kibana.
- Led modernization of CERN's Open Hardware Repository, migrating **500+ projects and 600+ users** and developing a new website.
- Automated infrastructure with Puppet, Foreman, Terraform, and Ansible.
- Built CI/CD pipelines with GitHub Actions, GitLab CI/CD, and Drone CI, and produced Docker images for testing, packaging, and simulation.
